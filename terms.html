Service description

FormSuite provides focused Google Workspace add-ons for Google Forms and Google Sheets workflows. The portfolio includes add-ons for response limits, confirmation emails, spreadsheet-backed choices, formulas, email merge, document generation, and approval workflows.

FormSuite is not a replacement for Google Workspace, a payment processor, a booking engine, a CRM, a strict inventory system, or a full enterprise approval platform.

Marketplace installation

The public installation path is Google Workspace Marketplace. This website is an install-first information and support hub. It is not currently a public checkout or subscription portal.

Testing responsibility

You are responsible for testing the add-on before using it with production workflows. Start with one copied form, one sample sheet row, one test recipient, one generated document, or one internal approval before sending real respondents or recipients through a workflow.
