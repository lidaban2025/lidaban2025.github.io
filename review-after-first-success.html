Before leaving a review

Use this page only after you personally installed the add-on and saw one useful result. If the add-on did not work yet, setup feedback is more helpful than a review.

  1. Confirm the add-on installed from Google Workspace Marketplace.
  2. Run one small first test in Google Forms or Google Sheets.
  3. Check the visible result: closed form, delivered email, copied formula, sent test message, generated Doc/PDF, or updated form choices.
  4. Open the matching Marketplace listing below.
  5. Leave an honest review that describes the first result in one or two sentences.
Do not leave a review for an add-on you did not use. Do not ask teammates, contractors, friends, or alternate accounts to leave reviews unless they independently installed and used the add-on for a real workflow.

Useful review wording

A review does not need to be long. The most helpful reviews are concrete and fair.

Short template

I used [add-on name] to [workflow]. My first test [specific result], and the setup step that mattered was [detail].

Examples of useful details

  • Closed a copied form at a test response cap.
  • Sent a respondent confirmation email from one public form submission.
  • Copied formulas into one new response row.
  • Generated one sample PDF from one Sheets row.

If the first run did not work

Please send setup feedback instead of guessing in a review. Include the add-on name, file type, test action, and the exact point where the result differed from what you expected.

Review policy guardrails

FormSuite asks only for honest reviews from real users after a successful first run. We do not offer rewards, discounts, payment, or any requested rating value for reviews.

Google Workspace Marketplace policies prohibit manipulating ratings, reviews, or install counts through illegitimate means such as fraudulent or incentivized downloads, reviews, and ratings. See Google's Marketplace program policies for details.
