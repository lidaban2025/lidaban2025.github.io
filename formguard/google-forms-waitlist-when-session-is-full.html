FAQ

Can Google Forms create a waitlist automatically?

Native Forms does not manage per-session waitlists by itself; use a separate form plus a capacity workflow.

Should the waitlist be in the same form?

A separate waitlist form is usually cleaner because it avoids confusing available choices.

What should the message say?

Tell users the session is full, link the waitlist, and set expectations about follow-up.
