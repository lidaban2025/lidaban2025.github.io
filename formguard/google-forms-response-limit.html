FormGuard guide

Set a Google Forms response limit and close the form after capacity

Use this when a form should stop after a total response cap, deadline, or capacity rule. Test it on a copied form before using it on a live signup.

Install FormGuardFormGuard overview

Quick answer

Use it when

A form has a hard limit.

Good fit for registrations, RSVP caps, class signup limits, and forms that should close after capacity.

Native path

Manual closing is fine for one-offs.

If the form is low volume, watch the response count and turn off Accepting responses yourself.

FormGuard path

Automate the close rule.

Set the limit, save the closed-form message, submit a test response, then check the public form.

Boundary

Keep the first test small.

Use a copied form and a tiny limit before applying the rule to production traffic.

Safe setup

  1. Copy the live Google Form.
  2. Install FormGuard and set a low test limit.
  3. Submit one or two test responses.
  4. Open the respondent preview and confirm the close message appears.
