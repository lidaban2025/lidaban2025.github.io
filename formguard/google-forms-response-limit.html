Set a Google Forms response limit and close the form after capacity

Google Forms can stop accepting responses manually, but teams often need a reliable capacity rule, close message, owner alert, and one safe test before launch.

Install FormGuardFormGuard overviewMore guides
Quick answer

Use a copied form first.

For one-off forms, watch the response count and turn off Accepting responses. For recurring signups, set a response limit in FormGuard, submit one test response, and confirm the public form closes with the right message.

Manual path

Native Google Forms controls are limited.

  1. Open the form Responses tab.
  2. Watch the count manually.
  3. Turn off Accepting responses when capacity is reached.
  4. Write a closed-form message for late visitors.
FormGuard path

Use it when capacity matters.

  1. Install FormGuard from Marketplace.
  2. Set the max response count or choice quota.
  3. Save the closed-form message.
  4. Submit one test response with a low limit before launch.
Best fit

Signups, events, and appointment forms.

Use this guide for workshop registration, class signups, RSVP caps, waitlist forms, role limits, or any form where overbooking creates support work.
