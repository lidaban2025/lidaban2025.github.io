Message template

Registration is currently full.
If you want to join the waitlist, use this form: [waitlist link].
We will contact waitlist respondents only if space becomes available.

Setup

  1. Create the main registration form.
  2. Create a separate waitlist form or contact page.
  3. Set the response cap in FormGuard.
  4. Paste the waitlist link into the closed message.
  5. Test once on a copied form before launch.
