Frequently Asked Questions

How do I create a template?

Create a Google Doc or Google Slides file and insert merge tags wherever you want personalized data to appear. Merge tags use the format {{field_name}}, where field_name matches a column header in your spreadsheet. For example, if your sheet has a column called "First Name", use {{First Name}} in your template.

What is the merge tag format?

Merge tags use double curly braces: {{field_name}}. The field name must exactly match the column header in your Google Sheet, including spaces and capitalization. Examples:

  • {{Name}} — inserts the value from the "Name" column
  • {{Email Address}} — inserts the value from the "Email Address" column
  • {{Company}} — inserts the value from the "Company" column

How do I run a merge?

  1. Open your Google Sheet containing the data you want to merge.
  2. Launch DocForge from the Extensions menu.
  3. Select your template document (Google Doc or Slides).
  4. Choose your output options (PDF, email, etc.).
  5. Click "Merge" to generate your documents.

My merge tags are not being replaced. What should I check?

  • Ensure the merge tag in your template exactly matches the column header in your sheet, including spaces and case.
  • Make sure you are using double curly braces: {{ and }}.
  • Check that there are no hidden characters or formatting differences. Try retyping the merge tag manually.
  • Verify the correct template is selected in the DocForge sidebar.

How does the AI template creation work?

The AI feature uses OpenAI to help you generate template content. You need to provide your own OpenAI API key in the DocForge settings. Once configured, you can describe the document you want to create and the AI will generate a template with appropriate merge tags based on your spreadsheet columns.

What are the usage limits?

Free users can perform up to 20 merges per month. Pro users get unlimited merges and up to 50 AI calls per month. Limits reset on the first day of each calendar month.

Can I merge into Google Slides?

Yes. Google Slides template support is available on the Pro plan. Create a Slides presentation with {{merge_tags}} and select it as your template in DocForge.
