Google Sheets add-on

Turn spreadsheet rows into polished Google Docs and PDFs.

DocForge helps teams generate letters, certificates, invoices, onboarding files, and custom documents from Google Sheets without exporting data to another tool. Choose a template, map fields, preview the merge, and generate files in a repeatable workflow.

What DocForge is for

DocForge is built for spreadsheet-first teams that need reliable document output without hand-editing every file.

Document generation from rows

Merge each row into a Google Doc or PDF using placeholder variables that match your sheet headers.

Template-based output

Use the same template across many recipients, records, or cases without rebuilding the document each time.

Optional email delivery

Send the generated document to recipients after the merge if your workflow includes outbound email.

Operator-friendly review

Run preflight before generating so mapping issues and missing configuration are caught before batch output starts.

Typical use cases

  • Offer letters, enrollment packets, and admissions documents from Google Sheets.
  • Certificates, confirmations, and internal documents from operational data.
  • Invoices, receipts, or customized PDFs that reuse the same structure with different row values.
  • Batch document generation for support, finance, HR, or academic operations teams.

Quick start

  1. Open the Google Sheet that contains the source data you want to merge.
  2. Prepare a Google Doc template using placeholders like {{Name}} and {{Email}}.
  3. Open DocForge from the Extensions menu and choose the source sheet and template.
  4. Review the field mapping, choose your output format, and set file naming rules.
  5. Run preflight, then generate the files or PDFs for the rows you want to process.
