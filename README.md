# FormSuite

FormSuite is a set of focused Google Workspace add-ons for Google Forms, Google Sheets, Gmail, Google Docs, and PDF workflows.

Website:

- https://formsuite.dev/

Demo videos:

- https://formsuite.dev/resources/demo-videos.html

Published add-ons:

- FormGuard: close Google Forms by response limit, deadline, schedule, or choice quota.
  - https://formsuite.dev/formguard/
- FormNotifier: send owner alerts and respondent confirmation emails from Google Forms.
  - https://formsuite.dev/formnotifier/
- FormCopy Pro: copy formulas, formats, and helper columns into new Google Forms response rows.
  - https://formsuite.dev/formcopy/
- FormMerge Pro: send personalized operational emails from Google Sheets rows.
  - https://formsuite.dev/formmerge/
- DocForge: generate Google Docs and PDFs from Google Sheets rows.
  - https://formsuite.dev/docforge/
