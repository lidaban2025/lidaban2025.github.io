What is inside

This is not a vague prompt bundle. It is a working starter pack centered on repeatable Google Workspace flows you can adapt for your own internal use, client delivery, or packaged automation offer.

Before you use this archive

This package is intentionally narrow. It is best for people who already know they want to automate around Google Forms, Gmail, Google Docs, and Sheets.

Good fit

  • You already work inside Google Workspace.
  • You want a starting point, not a full product.
  • You are comfortable making small adjustments yourself.

Not the best fit

  • You need a polished SaaS with account management.
  • You need Slack, Shopify, or CRM integrations on day one.
  • You need a full custom workflow without any operator setup.

FAQ

Do I get a hosted app?

No. This is a starter package with workflows, notes, and setup guidance. If you want a workflow configured for you, use the setup service.

Can I use it for client work?

Yes. The package is written to be a useful base for internal use or small client delivery, especially if you already work in Google Workspace.

What do I get?

3 Google Workspace workflow templates with setup notes and customization guidance.

Do I need an OpenAI API key?

Optional. The AI features work with or without an external API key.

Can I customize the templates?

Yes. All templates are Google Apps Script files you can modify.

Is installation support included?

This archived kit is not the current support path. Install a published add-on first, test one small workflow, then email the add-on name and first-run result.
