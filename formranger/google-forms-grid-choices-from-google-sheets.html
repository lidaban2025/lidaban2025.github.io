Grid choices

Populate Google Forms grid choices from Google Sheets

FormRanger updates supported Google Forms grid row choices from a selected Google Sheets column. Use it when row labels change in Sheets and should not be edited by hand.

Install FormRanger Setup guide Permissions

Quick answer

Put grid row labels in one clean Sheets column, map one grid question, run Preflight, click Update now, then check the public preview.

First-run path

  1. Create a copied form with one grid or checkbox grid question.
  2. Add two or three sample row labels in Sheets.
  3. Select the spreadsheet and source column in FormRanger.
  4. Run Preflight and Update now.
  5. Confirm the preview grid rows match the sheet.

Good fit

Evaluation forms

Criteria, people, modules, or locations.

Training feedback

Sessions, topics, or workshop names.

Checklists

Operational items that change between cycles.

Team surveys

Groups, offices, roles, or staff lists.

Boundary

Long grids can be hard to answer. FormRanger syncs row choices; it does not add live filtering, approvals, capacity, booking, or payment logic.
