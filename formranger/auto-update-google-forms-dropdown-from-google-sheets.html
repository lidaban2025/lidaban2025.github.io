Dropdown update

Auto update a Google Forms dropdown from Google Sheets

FormRanger updates a supported Google Forms dropdown from a selected Google Sheets column. Use it when your option list changes in Sheets and manual copy-paste is slowing you down.

Install FormRanger Setup guide Permissions

Quick answer

Keep choices in one clean Sheets column, map one dropdown in FormRanger, run Preflight, click Update now, then check the public preview.

First-run path

  1. Create a copied form with one dropdown question.
  2. Use a small Sheets column with Alpha and Beta.
  3. Select the spreadsheet and map the dropdown.
  4. Run Preflight and Update now.
  5. Confirm the preview dropdown shows the sheet values.

Good fit

Rosters

Names, teams, locations, or departments.

Events

Sessions, rooms, tracks, or category lists.

Catalogs

Products, services, package names, or item labels.

Cleanup

Sort and dedupe choices before respondents see them.

Boundary

Updating a dropdown is not the same as reserving a seat, locking inventory, collecting payment, or filtering choices live while a respondent answers.
