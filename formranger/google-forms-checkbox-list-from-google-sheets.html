Checkbox list

Populate a Google Forms checkbox list from Google Sheets

FormRanger updates a supported Google Forms checkbox question from a selected Google Sheets column. Use it when respondents can choose several options from a list maintained in Sheets.

Install FormRanger Setup guide Permissions Choices + limits

Quick answer

Use one clean Sheets column, map one checkbox question, run Preflight, click Update now, then check the public preview and a sample response.

First-run path

  1. Create a copied form with one checkbox question.
  2. Add three sample values in Sheets.
  3. Select the spreadsheet and source column in FormRanger.
  4. Run Preflight and Update now.
  5. Submit one sample response to inspect checkbox output.

Good fit

Interests

Topics, skills, tracks, or preference lists.

Roles

Volunteer roles, groups, committees, or services.

Programs

Support categories, program options, or internal lists.

Multi-select forms

Any case where more than one option may be selected.

Boundary

Checkbox syncing does not enforce capacity, reservations, stock counts, or payment. Use another control workflow when each option has a hard limit.
