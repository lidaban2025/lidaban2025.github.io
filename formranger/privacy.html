What FormRanger Pro Does

FormRanger Pro is an add-on for Google Forms that populates form question choices such as dropdowns, multiple choice, checkboxes, and grids from selected Google Sheets data. It keeps form choices aligned with the spreadsheet source you choose.

Data We Access

FormRanger Pro requests the following permissions to function:

  • Form structure (forms.currentonly) - We read and update supported question choices in the current form. We do not access other forms or form responses.
  • Selected spreadsheet file access (drive.file) - We access spreadsheet files that you explicitly select through the add-on flow. This is used to read source tabs, headers, and values needed for choice mappings.
  • User email and profile (userinfo.email, userinfo.profile) - We use account identity only for basic add-on operation, licensing, and support context.
  • Container UI (script.container.ui) - Used to display the add-on sidebar and picker dialogs within Google Forms.
  • Apps Script triggers (script.scriptapp) - Used to manage refresh/update automation for configured mappings.

Data We Store

FormRanger Pro stores only the minimum configuration required to run the add-on inside your Google account. This can include the selected spreadsheet ID, sheet name, column name, question ID, and mapping preferences such as sorting or duplicate removal. We do not copy entire spreadsheet datasets into external storage.

Data Storage

All user settings, such as question-to-column mappings and update preferences, are stored in Google Apps Script's Properties Service. This data remains within Google's infrastructure and is associated with your Google account and the add-on project.

We do not operate an external database for form or spreadsheet content. Spreadsheet values are read at runtime and used only to build the updated choice list inside the current form.

Data Protection

FormRanger Pro relies on Google's security controls for data in transit and at rest within Google Workspace and Apps Script. We do not independently export spreadsheet contents, store Google credentials, or maintain separate copies of linked spreadsheet data on our own servers.

  • Access is limited to spreadsheet files the user explicitly selects for a mapping.
  • Spreadsheet data is processed only for the choice-sync function requested by the user.
  • Only lightweight mapping configuration is persisted; raw spreadsheet rows are not retained outside Google services.
  • Support requests are handled through email; users should avoid sending unnecessary sensitive sheet data in support messages.

Data Sharing

We do not share, sell, rent, or disclose user data to third parties for advertising or marketing. Your form and spreadsheet data remains within the Google ecosystem while the add-on runs.

Data Retention

Settings are retained in Google Properties Service for as long as the add-on is installed or the configuration remains in the form. You can remove mappings in the add-on and uninstall the add-on from Google Workspace Marketplace.

Security

FormRanger Pro runs within Google's Apps Script platform, inheriting Google's authentication, transport security, and infrastructure protections. We do not maintain separate passwords, tokens, or customer-hosted databases for the spreadsheet content used by the add-on.
