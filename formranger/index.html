Boundary

Not a booking or inventory lock.

FormRanger updates Google Forms choices from selected spreadsheet data. It does not reserve seats, process payments, or change choices while a respondent is already filling out the form.

Frequently asked questions

Which question types work?

Dropdown, multiple choice, checkbox, multiple-choice grid, and checkbox grid choices.

When should I ask for a review?

Only after a real update succeeds and the public respondent preview shows the expected choices.

Can it handle dependent dropdowns?

Use FormRanger for spreadsheet-backed choice refresh before respondents use the form. It is not respondent-time conditional filtering.
