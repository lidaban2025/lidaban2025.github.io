Quick guide

Populate Google Forms choices from Google Sheets

FormRanger is a Google Forms add-on that updates dropdown, multiple-choice, checkbox, and grid choices from selected Google Sheets data. Use it when the source list already lives in Sheets and manual form edits are becoming slow or error-prone.

Install FormRanger Setup guide Permissions

Quick answer

Put the valid choices in one clean Google Sheets column, open the form, map one supported question in FormRanger, run Preflight, click Update now, then check the public preview.

First-run path

  1. Create a copied form with one choice question.
  2. Use a small Sheets column with values like Alpha and Beta.
  3. Select the spreadsheet and map the question in FormRanger.
  4. Run Preflight, Update now, and check the public preview.

Use it for

Dropdown lists

Departments, locations, products, sessions, or categories maintained in Sheets.

Multiple choice

One-answer questions where the option list changes before launch.

Checkbox lists

Multi-select questions backed by a spreadsheet list.

Grid rows

Row labels that should follow a selected Sheets column.

Do not use it as

FormRanger is not a booking system, payment workflow, inventory lock, or live dependent-dropdown engine. It updates supported form choices from selected spreadsheet data; it does not reserve availability while a respondent is filling out the form.
