First-run test

Test Google Forms dynamic choices before launch

FormRanger is a Google Forms add-on for updating supported form choices from selected Google Sheets data. Before sharing a live form, test one copied form, one question, and one small source column.

Quick answer

Use a copied form, one dropdown or multiple-choice question, and one Sheets column with Alpha and Beta. Map it in FormRanger, run Preflight, click Update now, then open the public preview.

Why this test matters

If installs are not turning into successful use, the first question is whether users can complete one safe first run. This page is the handoff from install anxiety to proof: one copied form, one source column, one supported question, one update, and one public preview check.

What this test proves

FormRanger can read the selected Sheet source, validate the mapping, update the selected question, and show the changed choices to respondents.

What it does not prove

It does not prove booking capacity, payment logic, live dependent dropdown behavior, or every mapping in a complex production form.

First-run path

  1. Create or copy a Google Form with one supported choice question.
  2. Create a Google Sheets column with two visible test choices.
  3. Select the spreadsheet, sheet tab, column, and question in FormRanger.
  4. Run Preflight, then use Update now.
  5. Open the public form preview and confirm the choices match the sheet.

Visual proof points

Run the same test after each important change

Repeat the copied-form check when you copy the form, change the Sheet tab, rename a source column, change a question type, or replace the public form link in a campaign. Those are the moments where a setup can look saved but no longer point at the thing respondents will use.

Before launch

Run the test before you post the form link, email respondents, or add the form to a website.

After source edits

Run it again after major list cleanup, renamed headers, formula changes, or tab changes.

Record the result for support and reviews

Write down the date of the test, the copied form name, the source tab, the source column, the mapped question, the Preflight result, the Update now result, and what the public preview showed. If the test fails, this becomes a support note. If it succeeds, it becomes the proof that a Marketplace review is based on real use rather than hope.

Pass criteria

Preflight passes

The selected file, tab, column, and question mapping are valid.

Update now reports updated

The result says a question was updated, not only saved.

Preview shows sheet values

The public form shows Alpha, Beta, or your real sample values.

Boundary is clear

The workflow syncs choices before use; it does not reserve inventory during a response.

If it fails

Check the sheet tab, column header, empty rows, duplicate values, question type, and Google permissions. If Update now reports skipped or error, do not send live respondents yet.

Preflight warning

Re-select the spreadsheet, confirm the sheet tab, and make sure the mapped question still exists in this form.

Update skipped

Look for empty source values, duplicate-only values, an unsupported question type, or a mapping saved against a copied form.

Preview unchanged

Open a fresh public preview window after the update. If it still shows old choices, send the update result and preview screenshot to support.

Wrong form changed

Confirm you opened FormRanger inside the same Google Form whose public link you are checking.

FAQ

Can I skip the copied-form test?

No. The copied-form test is the fastest way to separate install/setup issues from production form risk.

What should I include in a setup-help request?

Include the source tab name, source column header, mapped question type, Preflight result, Update now result, and whether the public preview changed.

When is it safe to leave a review?

After Preflight passes, Update now reports at least one updated question, and the public preview shows the expected Sheet values.

Does the test cover live capacity?

No. FormRanger updates choices; it does not reserve inventory, sell seats, or remove a choice while someone is filling out the form.

After the preview check

If Preflight passes, Update now reports at least one updated question, and the public preview shows the expected sheet values, that is the right moment to leave an honest Marketplace review. If any step is still unclear, use support first.
