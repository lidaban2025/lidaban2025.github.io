Quick start

Get running fast

  1. Open the form in Google Forms edit mode and launch FormRanger.
  2. Use Select spreadsheet so FormRanger can access the spreadsheet you explicitly choose.
  3. Pick the sheet tab and column you want to use as the live choice source.
  4. Save the configuration, run Preflight, and use Update now to verify the choices.
  5. Open the public form preview and confirm the visible choices match the source column before sharing the form.

First-run quality gate

  • Preflight should show at least one linked question and no issues.
  • Update now should report an updated question, not only a saved mapping.
  • If the update report says skipped or error, treat the setup as unfinished.
  • Do not ask respondents to use the form until the public preview shows the expected choices.

Permissions and setup notes

  • Google Forms access is used to update supported choice-based questions in the current form.
  • Drive file access is used for the spreadsheet file you select through Google Picker.
  • Container UI access displays the mapping sidebar and validation flow.
  • Start with one test question and one selected spreadsheet range before applying the setup to a live form.

FAQ

Common questions

Which question types are supported?

Multiple choice, dropdown, checkbox, grid, and checkbox grid questions.

Why does a mapping fail validation?

Typical causes are a removed form question, a renamed sheet tab, or a missing column header in the source spreadsheet.

Can I remove duplicate choices?

Yes. The mapping wizard includes a remove duplicates option before you save the link.
