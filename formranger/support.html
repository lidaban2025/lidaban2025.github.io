Quick start

Setup in 3 steps

  1. Open a copied Google Form and launch FormRanger.
  2. Select the spreadsheet, sheet tab, and column to use as choices.
  3. Run Preflight, click Update now, then preview the public form before sending it.

First-run quality gate

Do not launch until these pass

  • Preflight
    It says the mapping is valid and shows the expected mapping count.
  • Update report
    It says at least one question was updated. Skipped or error is not a successful run.
  • Public respondent preview
    The public form shows the choices from the selected Sheets column.
  • Review timing
    Only leave a Marketplace review after the setup succeeds in the public preview. The first-run checklist reveals that next step after all four checks and the preview confirmation pass.
FAQ

Common questions

Which questions work?

Dropdown, multiple choice, checkbox, multiple-choice grid, and checkbox grid questions.

Why did update fail?

Usually the form question, sheet tab, or source column was renamed or removed.

Can it remove duplicates?

Yes. Use the remove-duplicates option before saving the mapping.

Can FormRanger create dependent dropdowns?

Use FormRanger for owner-controlled Sheets-to-Forms choice refresh before respondents use the form. It is not respondent-time filtering where one answer instantly changes the next dropdown.

Is it a booking or inventory system?

No. FormRanger updates choices from a selected spreadsheet source. It does not reserve seats, process payments, or lock inventory while respondents are submitting.
