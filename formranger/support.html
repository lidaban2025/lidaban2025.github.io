Frequently Asked Questions

How do I set up FormRanger Pro?

  1. Open your Google Form in edit mode.
  2. Click the puzzle piece icon (Add-ons) in the top toolbar.
  3. Search for "FormRanger Pro" and click Install.
  4. Grant the required permissions when prompted.
  5. Go to Add-ons → FormRanger Pro → Configure.
  6. Select a form question, then choose the spreadsheet and column to pull choices from.
  7. Click Save. Choices will be populated immediately and updated automatically.

What question types are supported?

FormRanger Pro supports Multiple Choice, Dropdown, Checkbox, and Grid (both Multiple Choice Grid and Checkbox Grid) question types. Each can be linked to a different spreadsheet column.

How do I link a question to a spreadsheet column?

In the FormRanger Pro configuration panel, select the question you want to populate. Then choose the Google Sheets spreadsheet, the specific sheet tab, and the column containing your choice values. The add-on will read all non-empty values from that column and set them as the question's choices.

How often are choices updated?

By default, choices are updated every time the form is opened in edit mode. Pro users can also set up timed updates (every 1, 6, 12, or 24 hours) so choices stay current even without opening the form editor.

Can I remove duplicates from the choices?

Yes. Enable the "Remove Duplicates" option in the configuration for each linked question. The add-on will automatically filter out duplicate values from your source column before populating the form choices.

What happens if my spreadsheet is deleted or moved?

If the linked spreadsheet becomes inaccessible, FormRanger Pro will not be able to update choices. The form will retain the last set of choices that were successfully populated. You can reconfigure the add-on to point to a new spreadsheet at any time.

How do I uninstall FormRanger Pro?

Go to Add-ons → Manage add-ons, find FormRanger Pro, click the three-dot menu, and select "Remove". This will remove all stored settings and disable automatic choice updates for that form.
