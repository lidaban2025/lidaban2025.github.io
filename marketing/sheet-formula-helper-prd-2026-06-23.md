# Sheet Formula Helper PRD - 2026-06-23

## Objective

Build a focused browser tool for spreadsheet users who need to fix, explain, or generate formulas for Google Sheets and Excel.

## Positioning

Sheet Formula Helper is a formula-first utility. It is not a full spreadsheet automation suite, BI system, or data warehouse tool.

## Primary Jobs

- Fix a broken formula from the current formula, error, and intended result.
- Explain an existing formula in plain English.
- Generate a starter formula from a plain-language goal.

## First-Screen Product

The first version should show the working formula helper immediately, with three modes:

- Fix
- Explain
- Generate

The page should return copy-ready output and state assumptions clearly.

## Guardrails

- Do not ask users to paste confidential spreadsheets.
- Encourage one formula and one small example at a time.
- Show local fallback output when the AI endpoint is unavailable.
- Keep Google Sheets first, with Excel support as a secondary path.

## Success Signals

- Formula tool usage.
- Copy result usage.
- Support requests with one formula and one expected result.
- Search impressions for formula fixer, formula explainer, and formula generator queries.
