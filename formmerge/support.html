Quick start

Setup in 3 steps

  1. Open the Google Sheet and launch FormMerge.
  2. Write a template with merge tags that match your column headers.
  3. Send a test email to yourself before sending selected rows.

FAQ

Common questions

Why did a merge tag fail?

The tag usually does not match the exact sheet header.

Can I send all rows at once?

Start with selected rows and respect Gmail sending limits.

What should I include in support?

Include the header names, template snippet, and whether the test email worked.
