Quick start

Get running fast

  1. Open the response spreadsheet in Google Sheets and launch FormMerge.
  2. Choose the recipient email column or let the add-on auto-detect it.
  3. Write your subject and body template using merge tags from the sheet headers.
  4. Save, run Preflight, send yourself a test, then send all unsent rows when ready.

Permissions and reviewer notes

  • Google Sheets access to read response rows and write delivery status back to the sheet.
  • Mail access to send the messages you configure.
  • Container UI access for the spreadsheet sidebar only.

FAQ

Common questions

How do I insert merge tags?

Use the tag chips in the sidebar or type tags manually with double curly braces, such as {{Email}}.

Why did Preflight fail?

Most failures come from a missing recipient column, an empty email body, or low Gmail quota in the current account.

How do I see what failed during a batch send?

Batch send errors are written to FormMerge Error Report and the status column is updated in the sheet.
