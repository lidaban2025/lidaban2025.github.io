Features

Link Any Question Type

Connect Multiple Choice, Dropdown, Checkbox, and Grid questions to your spreadsheet data columns.

Auto-Update on Form Open

Choices are automatically refreshed every time the form is opened, so respondents always see the latest options.

Remove Duplicates

Link multiple form questions to different spreadsheet columns. Each question gets its own data source.
