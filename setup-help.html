Before sending

  1. Install the add-on from Google Workspace Marketplace.
  2. Open it in the right Google app: Forms for FormGuard/FormNotifier/FormRanger, Sheets for FormCopy/FormMerge/DocForge.
  3. Run one small test: one cap, one email, one copied formula, one generated file, or one synced choice list.
