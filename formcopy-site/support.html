Frequently Asked Questions

How do I set up FormCopy Pro?

  1. Open the Google Sheets spreadsheet linked to your Google Form responses.
  2. Click Extensions → Add-ons → Get add-ons.
  3. Search for "FormCopy Pro" and click Install.
  4. Grant the required permissions when prompted.
  5. Add your formulas to row 2 of the response sheet in any columns you need.
  6. Go to Extensions → FormCopy Pro → Enable to activate automatic copying.
  7. Every new form submission will now get your formulas automatically.

Where do I put my formulas?

Place your formulas in row 2 of the form response sheet, in the columns where you want them. FormCopy Pro detects these formulas and copies them (with adjusted row references) into every new response row. You can use any formula: SUM, VLOOKUP, IF, CONCATENATE, etc.

What if I already have responses without formulas?

Use the Manual Batch Run feature. Go to Extensions → FormCopy Pro → Batch Run. This will apply the formulas from row 2 to all existing response rows that are missing them.

Will it overwrite my existing data?

FormCopy Pro only writes formulas to cells that correspond to formula columns in row 2. It will not overwrite cells in columns where row 2 contains plain values (not formulas). However, if a formula column already has data in a response row, the formula will replace it.

Does it work with VLOOKUP and other reference formulas?

Yes. FormCopy Pro works with any formula, including VLOOKUP, INDEX/MATCH, IMPORTRANGE, and more. Row references within formulas are automatically adjusted for each new row.

How do I check my usage?

Go to Extensions → FormCopy Pro → Settings. Your current submission count and remaining quota for the month are displayed. The Free tier allows 50 submissions per month. Usage resets on the first day of each calendar month.

How do I uninstall FormCopy Pro?

Go to Extensions → Add-ons → Manage add-ons, find FormCopy Pro, click the three-dot menu, and select "Remove". This will remove all stored settings and disable automatic formula copying.
