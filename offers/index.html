This page is for users who already installed or are about to install a published FormSuite add-on. New visitors should go to Marketplace first, test one small workflow, and only then contact FormSuite for setup feedback or support.

What happens after contact

Keep expectations crisp. Users should know this is a support and feedback path after install.

First-run feedback

Reply with a short confirmation, clarify the setup path, and ask for a Marketplace review only if the workflow succeeded.

Setup blockers

Use the reported add-on, workflow, and screenshot to identify whether the issue is documentation, UX, permissions, or a product bug.

Setup clarification

Only discuss implementation questions after the user has a real workflow and understands what the add-on does after installation.

Frequently asked questions

Which path should I pick?

Install first if you have not tried a published add-on. Send feedback if you completed a first run. Ask setup questions only after the workflow is clear.

Can I ask setup questions later?

Yes. Start with a Marketplace install and a small test, then ask for setup help if the workflow is worth implementing.

Is anything required before install?

No. Published add-ons should be installed from Google Workspace Marketplace first during launch mode.

How does contact work?

Email the add-on name, workflow, and first-run result. That keeps support and future setup conversations grounded in a real install.
