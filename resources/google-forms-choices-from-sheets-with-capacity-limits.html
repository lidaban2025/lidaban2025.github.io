Quick answer

Sync the list first, then test the limit. FormRanger updates supported dropdown, multiple-choice, checkbox, and grid choices from a selected Sheets column. FormGuard controls when the form closes or when an option should stop accepting more selections.

Safe first-run path.

Keep the first run small.

1

Sync choices

Use a copied form, one question, and Alpha/Beta in Sheets. Run FormRanger Preflight and Update now.

2

Set the limit

Use FormGuard on the same copied form with a tiny cap or one choice quota.

3

Preview before launch

Open the public preview, submit samples, and confirm the choices plus close behavior.

Important boundary

Do not treat a visible Google Forms option as a guaranteed live reservation. For booking, payment, inventory locking, or high-demand seat allocation, use a buffer, waitlist, manual review, or a purpose-built system.
