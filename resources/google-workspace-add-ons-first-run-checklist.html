Open FormRanger checklist

Universal first-run rule

  1. Use copied data or one internal test file.
  2. Install from Google Workspace Marketplace.
  3. Configure the smallest possible version of the workflow.
  4. Run the product's Preflight or test action.
  5. Confirm the visible result in the form, email, Sheet, Doc, or PDF.
  6. Use support before production if the result does not match.
  7. Ask for or leave an honest Marketplace review only after the test works.
