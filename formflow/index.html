Frequently asked questions

How should I start?

Install FormFlow, open it from a Google Form, select the linked response spreadsheet, add one approver, save, and run Preflight.

What workflow should I test first?

Use one approval process with known reviewers, clear decisions, and a linked Google Sheets response file.

Do I need a separate approval system?

The goal is to keep intake in Google Forms while adding a structured approval layer around it.

Is the current OAuth app verification approved?

Yes. Google approved the current FormFlow OAuth App Verification request for its Apps Script execution, email sending, and container UI scopes on May 26, 2026. The public Marketplace install path and first-run Preflight smoke were verified on May 27, 2026.
