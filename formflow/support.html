Quick start

First-run test path

  1. Use a copied or internal Google Form, not the live production form.
  2. Select the response spreadsheet linked to that form.
  3. Add one internal approver and save the configuration.
  4. Run Preflight and resolve any warning before launch.
  5. Submit one test response, then confirm the approval email, approve/reject link, and linked Sheet status.

Need help?

Send these four checks

  • Product
    FormFlow
  • Preflight
    Did Preflight pass, fail, or show a warning?
  • Approval email
    Did the reviewer email arrive in the intended inbox?
  • Decision link
    Did approve or reject work for the intended reviewer?
  • Sheet status
    Did the linked response sheet update after the decision?
  • Screenshot
    Include the sidebar, email, decision page, or Sheet status that failed.
  • Contact
    Send a FormFlow setup note or use FormSuite support.

FAQ

Common questions

Is this for complex enterprise workflows?

No. It is for lightweight Forms-to-Sheets approval flows.

What should I test?

Check that Preflight passes, the approval email arrives, the approve/reject link works, and the linked Google Sheet status updates.

What if reviewers do not get emails?

Check reviewer addresses, Gmail limits, and spam folders.
