Quick start

Get running fast

  1. Open the Google Form in edit mode and launch FormFlow from Add-ons.
  2. Add one or more approval levels and list the approver email addresses for each stage.
  3. Adjust subject lines, button text, and reminder settings if needed.
  4. Save the configuration and run Preflight before using the workflow live.

Permissions and reviewer notes

  • Google Forms access to read submissions and maintain approval state.
  • Google Sheets access only for the linked response sheet and approval status tracking.
  • Gmail or MailApp access to send approval and submitter notifications.

FAQ

Common questions

How do approvals know which response to update?

FormFlow stores approval metadata against the active form and its linked response sheet, then updates status records as decisions arrive.

Can I require comments before rejecting?

Yes. Turn on the require comment option in Advanced Options.

What should I do before inviting reviewers?

Run Preflight, send one internal test submission, and confirm each approver email can receive messages from the workflow.
