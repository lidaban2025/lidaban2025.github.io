Permissions

Why DocForge asks for Google permissions

DocForge runs inside Google Sheets. It reads row data, uses a Google Docs template, and creates Docs or PDFs in your Google Drive according to the setup you configure.

Install DocForgeOpen setup guide

Permission mapping

ScopePlain-English use
spreadsheets.currentonlyLets DocForge read source rows and write output status back to the spreadsheet you open.
documentsLets it read template structure and create or update generated Google Docs.
drive.fileLets it create and manage files that DocForge creates or opens through the add-on workflow.
script.container.uiLets it display the sidebar inside Google Sheets.
userinfo.emailLets it identify the installing account for file ownership context and support context.

What DocForge does not do

  • It is not a full document management system, contract lifecycle platform, or approval platform.
  • It does not sell spreadsheet, template, or generated document data.
  • It is designed for template-based document generation from rows you control.

Uninstall and support

You can remove DocForge from Google Workspace Marketplace's Manage Apps page. Generated files remain in your Google Drive unless you delete them.
