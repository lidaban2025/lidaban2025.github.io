Frequently asked questions

Can DocForge generate PDFs?

Yes. DocForge can generate Google Docs and PDF output from spreadsheet rows.

Do I need a Google Docs template?

Yes. The template defines the layout, and the sheet columns provide the row data.

What should I test first?

Generate one document from one sample row, inspect the output, then run a small batch.
