Privacy Policy

Last updated: March 19, 2026

What DocForge Does

DocForge is a Google Sheets add-on that merges data from your spreadsheets into Google Docs and Google Slides templates, generating personalized documents at scale.

Data We Access

DocForge requests the following Google API scopes to function:

  • Spreadsheet data (spreadsheets.currentonly) — reads data from the active spreadsheet to populate merge fields in your templates.
  • Google Documents (documents) — reads and writes Google Docs templates and generates merged output documents.
  • Google Presentations (presentations) — reads and writes Google Slides templates and generates merged output presentations.
  • Google Drive (drive) — manages files including creating merged documents, exporting PDFs, and organizing output in Drive folders.
  • Email (script.send_mail via MailApp) — sends merged documents as email attachments when you use the email feature.

How We Use Your Data

Your data is used solely to perform document merges as instructed by you. We do not collect, store, share, or sell your data. All processing happens within your Google account environment.

AI Feature (Optional)

DocForge includes an optional AI-powered template creation feature. When you explicitly trigger this feature:

  • Data is sent to OpenAI's API using your own API key that you provide.
  • No data is sent to OpenAI unless you explicitly initiate an AI action.
  • Your API key is stored securely in Google Apps Script Properties Service and is never shared with us or any third party.

Data Storage

DocForge stores add-on settings (such as preferences and your OpenAI API key) using Google Apps Script Properties Service, which is scoped to your Google account. No data is transmitted to or stored on external servers, with the sole exception of the optional AI feature described above.

Third-Party Services

  • Google Workspace APIs — for core merge functionality.
  • OpenAI API — only when you explicitly use the AI feature with your own API key.

Data Retention and Deletion

DocForge does not retain any of your data beyond the current session. Merged documents are created in your own Google Drive. To remove all stored settings, uninstall the add-on from your Google account.

Contact

If you have questions about this privacy policy, please contact us at FormSuite support.
