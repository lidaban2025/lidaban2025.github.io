Quick start

Setup in 3 steps

  1. Open the source Google Sheet and launch DocForge.
  2. Choose a Google Docs template and map placeholders to sheet columns.
  3. Generate one test document before running a batch.

FAQ

Common questions

Why is a placeholder blank?

The placeholder usually does not match a sheet header or the row cell is empty.

Should I generate PDFs first?

Generate one Google Doc preview first, then export PDF after the mapping looks right.

What should I send support?

Include the placeholder names, column headers, and whether the test document was created.
