Quick start

Get running fast

  1. Open the response spreadsheet in Google Sheets.
  2. Launch DocForge from Extensions and choose the sheet tab you want to merge from.
  3. Select or create a Google Docs or Slides template with matching merge tags.
  4. Run Preflight, review warnings, then generate documents or PDFs.

Permissions and reviewer notes

  • Google Sheets access to read the selected source sheet.
  • Google Drive and Docs access to open your chosen template and create output files.
  • Optional Gmail access only when you choose email delivery in the workflow.

FAQ

Common questions

My template does not appear in the sidebar.

Create the template in Google Docs or Google Slides first, then reopen the sidebar or confirm the file lives in the same Google account you are reviewing with.

Merge tags are not being replaced.

Make sure each tag exactly matches a column header in the source sheet, including spaces and capitalization.

Preflight says a required field is missing.

Check the selected sheet tab and confirm the template fields still map to live column headers in row 1.
