DocForge guide

Turn Google Sheets rows into PDF documents

Use this when one spreadsheet row should become a Google Doc, PDF, certificate, letter, invoice, or repeatable document from a template.

Install DocForge DocForge overview

Quick answer

Row

One record per row.

Keep stable sheet headers and clean sample data.

Template

Use a Docs template.

Match template fields to sheet columns before generating files.

Test

Generate one PDF first.

Inspect layout, file name, destination, and merged values.

Boundary

Not a full DMS.

Use it for repeatable generation, not document lifecycle management.

Safe setup

  1. Keep one record per row in Google Sheets.
  2. Create a Google Docs template with matching fields.
  3. Open DocForge from the Google Sheets Extensions menu.
  4. Generate one test document before running a batch.
